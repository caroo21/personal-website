# Personal Website

Portfolio website built with vanilla HTML, CSS, and JavaScript.

🌐 **Live:** [carolinewendlandt.com](https://www.carolinewendlandt.com)

## Stack
- HTML5, CSS3, vanilla JavaScript
- Deployed via GitHub Pages

## Structure
```
├── index.html
├── contact.html
├── projects.html
├── life-stations.html
├── index.css
├── index.js
├── favicon.ico
├── CNAME
├── robots.txt
├── sitemap.xml
├── .gitignore
├── README.md            
└── assets/
```

## Run Locally
Clone the repo and open `index.html` in a browser — no build step needed.

## Deployment
Pushed to `main` → auto-deployed via GitHub Pages.
